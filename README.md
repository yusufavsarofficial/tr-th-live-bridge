# Pingle

Pingle, mobil öncelikli özel iki kişilik sohbet uygulamasıdır.

## Özellikler

- 2 kişilik oda limiti
- Otomatik çift dilli mesaj desteği
  - Türkçe yazınca: Türkçe + Tayca
  - Tayca yazınca: Tayca + Türkçe
  - Emoji mesajları çeviriye sokmadan olduğu gibi gönderilir
- Sesli ve görüntülü görüşme
- Düşük veri kullanımı için sınırlı WebRTC bitrate ve mobil video çözünürlüğü
- Görüşme sırasında küçültülebilen arama paneli
- Emoji gönderme
- Profil resmi URL desteği
- Sohbet geçmişi dışa aktarma ve temizleme API'leri
- Masaüstü modu ve mobil uyumlu web arayüzü

## Hemen Çalıştır

```bash
npm install
npm start
```

Aç:

- http://localhost:3000

## Masaüstü Modu

```bash
npm run desktop
```

## Test
